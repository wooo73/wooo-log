# 맵과 방향벡터

맵은 하나의 그래프입니다. 예를 들어 3 \* 3 맵이며 1은 갈 수 있는 지역, 0은 갈 수 없는 지역이며 4방향으로 탐색이 가능하다고 하면 다음과 같은 그래프가 되는 셈입니다.
보통 이 맵을 기준으로 4가지 방향으로 탐색해나아가라고 합니다. 4가지 방향은 보통 위, 아래, 오른쪽, 왼쪽으로 주어집니다.
위쪽부터 시계방향으로 탐색한다고 가정하면 -1, 0 / 0, 1 / 1, 0 / 0, -1 을 더해가며 탐색하게 됩니다.

코드 구현

```c++
const int dy[] = {-1, 0, 1, 0};
const int dx[] = {0, 1, 0, -1};
for(int i = 0; i < 4; i++){
    ny = y + dy[i];
    nx = x + dx[i];
}
```

{0, 0}좌표에서 dy, dx를 만들어 4방향(위, 오른쪽, 아래, 왼쪽)을 탐색하며 좌표를 출력하시오

```js
function main() {
  // 상하좌우 좌표
  let dy = [-1, 0, 1, 0];
  let dx = [0, 1, 0, -1];

  let x = 0,
    y = 0;

  for (let i = 0; i < 4; i++) {
    y += dy[i];
    console.log(dy[i], ":", dx[i]);
  }
}

main();

// 8가지 방향
function main2() {
  const dy = [-1, -1, 0, 1, 1, 1, 0, -1];
  const dx = [0, 1, 1, 1, 0, -1, -1, -1];

  let x,
    y = 0;

  for (let i = 0; i < 8; i++) {
    x += dx[i];
    y += dy[i];
    console.log(dy[i], ":", dx[i]);
  }
}

main2();
```
